## MegaClient 1.9.0 Update Log

- Improved compatibility with legitimate mods, including **Simple Voice Chat**, **C2ME**, native-library mods, performance mods and compatibility addons.

- Improved launch protection so normal mods are no longer blocked because they use native files, temporary folders or similar compatibility references.

- Improved mod and dependency selection so installed content better matches the selected **Minecraft version** and **mod loader**.

- Improved Fabric Loader handling so compatible profiles are not unnecessarily changed every time the game launches.

- Added fully automatic launcher update scanning on startup, throughout the session, after sleep and when returning to MegaClient.

- Launcher updates now download automatically and clearly show their progress before asking to restart.

- Added optional **Discord activity status** for browsing the launcher, starting Minecraft and playing on an instance or server.

- Improved the skin and cape viewer with a sharper character render, complete outer layers, smoother controls and a better 3D cape preview.

- Improved launch performance by moving heavier mod checks away from the interface, reducing repeated Windows scans and optimising live console output.

- Improved startup and long modded launches to reduce freezing, false failures and unnecessary **Not Responding** behaviour.

- Protected MegaClient installation files are now hidden from normal folder views and appear when **Hidden items** is enabled in Windows Explorer.

- Added further launcher stability, loading, error handling, spacing and interface polish.
