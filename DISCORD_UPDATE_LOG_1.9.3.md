## MegaClient 1.9.3 Update Log

- Fixed issues with uploading and applying Minecraft skins.

- Fixed cape selection so equipped and hidden capes update more reliably.

- Improved handling when Minecraft Services returns delayed or incomplete skin and cape information.

- Prevented duplicate skin upload windows and overlapping cosmetic actions.

- Added clearer loading states while choosing a skin, uploading it, equipping a cape or hiding a cape.

- Improved skin file validation and made upload errors easier to understand.

- Improved Microsoft session recovery when a skin or cape request needs a refreshed login token.

- Added additional profile caching and reliability improvements so owned skins and capes remain visible after changes.
